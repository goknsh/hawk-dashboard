Free website uptime monitoring

Hawk is a free website uptime monitor that notifies you when your website is down or the latency is high.

Features

Notifications
Get notified by email anytime your website goes down, the latency is high or if your website's SSL expires (SSL expiry notification coming soon).
Regular Checks
Hawk will check if your website is down, website's latency and if your SSL is valid every five minutes.
Extensive Stats
Hawk collects statistics like latency at a certain speed, nameserver lookup time, data downloaded and more.
Many Locations
Hawk can monitor your websites from several locations, so you'll get different stats for each location.

How does it work?

Free Hosting
Currently, Hawk runs on free hosting services like heliohost.org, which allows us to keep Hawk free for you.
PHP & cURL
Hawk ulilizes PHP to make cURL requests and it sorts through the response while organizing the data.
MySQL
MySQL is the database that Hawk uses to store data and other statistics for each of your websites. We archive data every week, but also post copies of a JSON file of the archived data.
