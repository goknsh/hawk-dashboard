Free website uptime monitoring

Ping is a free website uptime monitor that notifies you when your website is down or the latency is high.

Features

Notifications
Get notified by email anytime your website goes down, the latency is high or if your website's SSL expires (SSL expiry notification coming soon).
Regular Checks
Ping will check if your website is down, website's latency and if your SSL is valid every five minutes.
Extensive Stats
Ping collects statistics like latency at a certain speed, nameserver lookup time, data downloaded and more.
Many Locations
Ping can monitor your websites from several locations, so you'll get different stats for each location.

How does it work?

Free Hosting
Currently, Ping runs on free hosting services like heliohost.org, which allows us to keep Ping free for you.
PHP & cURL
Ping ulilizes PHP to make cURL requests and it sorts through the response while organizing the data.
MySQL
MySQL is the database that Ping uses to store data and other statistics for each of your websites. We archive data every week, but also post copies of a JSON file of the archived data.
