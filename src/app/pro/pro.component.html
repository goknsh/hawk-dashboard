About Hawk’s Pro plan

Hawk Pro:

We are considering introducing a Pro version of Hawk. Pro will be based on a pay-for-what-you-use at: $0.00002 per interval per check. But if we get a sponsor, it will be free for everone and all of Hawk will be open sourced. Anyways, Pro will have features like:

  • Custom intervals for checks
  • Collect data from real browsers, like:
    • Image Compare for each check
    • Saves Screenshot when an outage occurs
    • Super-detailed analytics, with waterfall charts for each check
    • SMS Notifications
  • More regions
  • AWS Servers
  • DNS Logs for each check
  • PageFire: Basically Google Analytics, but powered by Hawk
  • Insights: Analyzes page to help you recognize things you can fix
  • GitHub commit ingeration with graph
  • Custom validators @ $1/validator/month
    • Custom Errors for HTTP codes
    • DNS rollback when down
    • Show custom code when down
    • GitHub commit rollback when down
    • Webhooks
  • DoThis @ $1/month: allows you to tell the browser what to do; and if it fails, we’ll alert you. One custom validator included.
  • Monitor resources (CSS, JS, PNG, JPG, GIF) for each check.

We haven’t started developement yet, but we have a plan. We’ll start developement if we get enough intrest. Anyhow, here’s the good news: there will be an alpha and beta for Pro, which you can test for free. So please express your intrest.
